 PROJECT

01

memoryGame

티니핑 캐릭터로 만든 memory game은 JavaScript 학습을 위해 만든
개인 프로젝트로 짝이 맞는 카드를 찾는 간단한 게임입니다.
게임 방식은 캐릭터가 있는 카드의 앞면이 화면에 3초 동안 보여지고
3초 뒤, 다시 뒤집힌 카드의 위치를 기억해 짝이 맞는 카드를 찾아야 합니다.
이미지 클릭 이벤트를 통해 사용자가 선택한 두 이미지를 비교하여
일치 여부를 판단하는 로직은 바닐라 JavaScript로 구현하였고,
이미지 클릭 시 .flipped 클래스를 추가하여 3D 회전 효과를 적용해
카드가 뒤집히는 느낌은 CSS 애니메이션으로 구현하였습니다.
